# 06 - 前端页面设计

> 前端采用 **React 18 + Vite 6** 构建 SPA 应用，UI 框架为 **Tailwind CSS + shadcn/ui**，路由使用 **React Router v7**。

## 6.1 设计系统

### 6.1.1 设计原则

| 原则 | 说明 |
|------|------|
| 简洁优先 | 降低认知负荷，突出核心操作 |
| 一致性 | 全站统一的组件、色彩、间距 |
| 响应式 | 桌面优先，适配平板和移动端 |
| 可访问性 | WCAG 2.1 AA 标准 |

### 6.1.2 色彩体系

```css
/* 品牌色 */
--primary:     hsl(220, 90%, 56%);  /* 蓝色，主操作 */
--primary-fg:  hsl(0, 0%, 100%);

/* 语义色 */
--success:     hsl(142, 71%, 45%);  /* 绿色，成功/通过 */
--warning:     hsl(38, 92%, 50%);   /* 橙色，警告/待审核 */
--destructive: hsl(0, 84%, 60%);    /* 红色，错误/危险 */
--info:        hsl(199, 89%, 48%);  /* 浅蓝，信息 */

/* 信用等级色 */
--level-seed:        hsl(120, 40%, 55%);  /* 🌱 绿色 */
--level-craftsman:   hsl(30, 60%, 50%);   /* 🔧 橙色 */
--level-expert:      hsl(45, 100%, 50%);  /* ⚡ 金色 */
--level-master:      hsl(270, 60%, 55%);  /* 🏆 紫色 */
--level-grandmaster: hsl(0, 80%, 55%);    /* 👑 红色 */

/* 暗色模式（预留） */
/* 使用 Tailwind dark: 前缀，通过 Zustand 管理主题状态 */
```

### 6.1.3 排版

```
字体栈：
├── 中文：system-ui, "PingFang SC", "Microsoft YaHei"
├── 英文：Inter, system-ui
└── 代码：JetBrains Mono, "Fira Code", monospace

字号层级：
├── 页面标题：text-3xl (30px) / font-bold
├── 区块标题：text-xl (20px) / font-semibold
├── 正文：text-base (16px)
├── 辅助文字：text-sm (14px) / text-muted-foreground
└── 标签/徽章：text-xs (12px)
```

/* Tailwind 默认断点 */

| 管理后台 | 左侧边栏 | 收起侧边栏 | 底部导航 |
