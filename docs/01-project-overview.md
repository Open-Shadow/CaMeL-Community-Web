## 1.2 项目愿景

构建一个以 AI Skill（Prompt/Agent）为核心资产的社区平台，通过 **Skill Marketplace（技能市场）**、**Bounty Board（悬赏任务板）**、**Workshop（知识工坊）** 三大模块，形成 "创作 → 交易 → 沉淀" 的自增强循环。

---

## 1.3 核心用户画像

| 用户类型 | 特征 | 核心需求 |
|----------|------|----------|
| 消费者 | 有 AI 使用需求，不擅长 Prompt 工程 | 快速找到好用的 Skill，高效完成任务 |
| 创作者 | Prompt 高手，技术能力强 | 通过创作获取 $ 收入和声望 |
| 需求方 | 有明确定制需求的企业/个人 | 通过悬赏找到合适的人完成任务 |
| 接单者 | 有技术能力，愿意接活的开发者 | 通过完成悬赏任务赚取 $ |

## 1.4 经济系统设计

### 1.4.1 双轴体系

系统采用 **$ 额度 + 信用分** 双轴体系，不设双 Token：

| 维度 | $ 额度（硬通货） | 信用分（声望值） |
|------|-------------------|------------------|
| 性质 | 可充值、可消费、可赚取 | 不可充值、不可转让、不可兑换 |
| 获取 | 充值 / 出售 Skill / 完成悬赏 / 邀请奖励 | 社区行为积累 |
| 用途 | 调用 API、购买 Skill、发布悬赏 | 解锁特权、降低使用成本 |
| 核心逻辑 | 所见即所得，无汇率 | 贡献越多，用得越便宜 |

### 1.4.2 信用分等级体系

| 等级 | 图标 | 信用分要求 | API 倍率折扣 | 核心特权 |
|------|------|-----------|-------------|---------|
| 新芽 | 🌱 | 0 ~ 99 | 1.0x（无折扣） | 基础功能 |
| 工匠 | 🔧 | 100 ~ 499 | 0.95x | 可发布 Skill |
| 专家 | ⚡ | 500 ~ 1999 | 0.90x | 可参与仲裁 |
| 大师 | 🏆 | 2000 ~ 4999 | 0.85x | VIP 群组 |
| 宗师 | 👑 | 5000+ | 0.80x | 内测优先权 |

### 1.4.3 信用分获取途径

| 行为 | 信用分 | 备注 |
|------|--------|------|
| 完善个人资料 | +10 | 一次性 |
| 每日登录 | +1 | 每日上限 1 |
| 发布 Skill | +15 | 审核通过后 |
| Skill 被调用（每 100 次） | +5 | - |
| 完成悬赏任务 | +20 | 验收通过后 |
| 发布 Workshop 文章 | +10 | - |
| 文章被加精 | +20 | - |
| 打赏他人 | +2/$ | - |
| 邀请新用户 | +30 | 被邀请者首充后 |

### 1.4.4 信用分扣减事件

| 行为 | 扣减 | 备注 |
|------|------|------|
| 悬赏任务超时未提交 | -15 | 累计 3 次禁止接单 |
| 发布低质量内容（被删除） | -10 | - |
| 评价被举报且确认恶意 | -20 | - |
| 仲裁败诉 | -10 | 悬赏相关 |
| 信用分 < 30 | - | 悬赏板冻结 30 天 |

## 1.5 三模块协同关系

```
            Skill Marketplace
           /        ↑         \
          /   创作者写教程发到    \
         /    Workshop 引流     \
        ↓                       ↓
   Workshop ◄────────────► Bounty Board
   (知识沉淀)   用户看教程后    (需求撮合)
                发现需求，
                去悬赏板发任务
```

**协同场景闭环**：

1. 用户 A 在 Workshop 看到教程
2. A 想要定制版本 → 去 Bounty Board 发悬赏
3. 用户 B 接单完成 → 赚取 $
4. B 把成果打包成 Skill 上架 Marketplace
5. 更多用户使用 Skill → B 获得被动收入 + 信用分
6. B 写新 Workshop 文章 → 循环继续

**交叉引用**：
- Workshop 文章可嵌入 Skill 卡片
- Bounty 完成后提示上架 Skill
- Skill 详情页显示关联 Workshop 文章
- 用户主页统一展示所有模块贡献

## 1.6 邀请裂变机制

| 事件 | 邀请者奖励 | 被邀请者奖励 | 触发条件 |
|------|-----------|-------------|---------|
| 注册成功 | +10 信用分 | +10 信用分 | 即时 |
| 首次充值 ≥ $1 | +$0.50 + 30 信用分 | - | 延迟发放 |
| 首月消费 ≥ $5 | +$1.00 + 50 信用分 | - | 延迟发放 |

**反刷机制**：
- 同一 IP/设备注册多个账号仅首个生效
- 被邀请者 7 天内无 API 调用 → 邀请关系作废
- 单用户每月邀请奖励上限：$10 + 200 信用分

## 1.7 项目范围（Scope）

### In Scope

- 用户系统（注册/登录/个人中心/信用分）
- Skill Marketplace（上传/审核/浏览/搜索/交易）
- Bounty Board（发布/接单/交付/结算/仲裁）
- Workshop（发布/浏览/投票/打赏/加精）
- $ 充值与站内结算
- 邀请裂变
- 管理后台（审核/版主/数据看板）

### Out of Scope (v1)

- 移动端原生 App
- 第三方 API 开放平台
- $ 提现功能
- 即时通讯（IM）
- 多语言国际化（预留架构）
